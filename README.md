# Gong

Gong is a simple Pong clone implemented in Go, a classic arcade game where players control paddles to hit a ball back and forth. 

## Installation

1. Make sure you have **Go v1.13** or later installed on your system.
2. Clone the Gong repository to your local machine using Git:

```bash
git clone https://github.com/finnbyte/Gong.git
```

3. Navigate to the directory where Gong is cloned:

```bash
cd Gong
```

4. Run the following command to build and start the game, with dependencies automatically installed:

```bash
go run main.go
```
